Web app that provides a query box and a search button and then calls 
the Places Library for Google Maps 
(https://developers.google.com/maps/documentation/javascript/places). 
Format the results to give a good user experience. 

-index.html is the only page.
- works better in Firefox/IE
- Desktop App and positions the map to its center up on reducing/expanding screen.
- Opening this html would land on a generic map with a search input field.
- any text/location can be searched. Closest match would be retrieved and displayed on the map.
- Search can be specific or generic search, accordingly either one or multiple findings would be returned as result.
